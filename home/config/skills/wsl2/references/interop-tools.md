# Common Windows Interop Tools

- **Explorer**: `explorer.exe .` - Open current folder in Windows Explorer.
- **VS Code**: `code .` - Open current folder in VS Code.
- **PowerShell**: `powershell.exe` - Launch Windows PowerShell.
- **CMD**: `cmd.exe` - Launch Windows Command Prompt.
- **Task Manager**: `taskmgr.exe` - Launch Windows Task Manager.
- **Wsl-Open**: `wsl-open <file>` - Open file in default Windows application.
- **Systeminfo**: `systeminfo.exe` - View Windows system details.
- **Ipconfig**: `ipconfig.exe` - View Windows network configuration.
- **Clip**: `clip.exe` - Pipe output to Windows clipboard.
